# Databricks Data Engineer Professional Certification Study Plan (2025/2026)

## Overview
This plan covers the 10 domains from the September 2025 exam guide. 

## Schedule

### Week 1: Professional Development & Tooling
- **Day 0:** [x] Environment Setup (M1 Mac, CLI, VS Code, pyenv, Azure OAuth)
- **Day 1:** [x] Python Project Structure & Databricks Asset Bundles (DABs)
- **Day 2:** [x] Dependency Management (Wheels, PyPI) & CLI/REST APIs
- **Day 3:** [x] Advanced UDFs (Pandas/Python UDFs)
- **Day 4:** [x] Unit Testing & Integration Testing (assertDataFrameEqual, Debugging)
- **Day 5:** CI/CD Integration with Git Folders & DABs

### Week 2: Scalable Data Pipelines (Lakeflow & Ingestion)
- **Day 1:** Lakeflow Declarative Pipelines (DLT) & Auto Loader
- **Day 2:** Streaming Tables vs. Materialized Views
- **Day 3:** CDC with APPLY CHANGES API
- **Day 4:** Control Flow in Pipelines & Ingestion Formats (Avro, XML, Binary)
- **Day 5:** Spark Structured Streaming vs. Lakeflow

### Week 3: Governance, Security & Sharing
- **Day 1:** Unity Catalog Managed Tables & Permission Inheritance
- **Day 2:** Delta Sharing (D2D and D2O)
- **Day 3:** Lakehouse Federation
- **Day 4:** Row Filters & Column Masks
- **Day 5:** Data Anonymization (Hashing, Tokenization) & PII Masking

### Week 4: Performance, Monitoring & Optimization
- **Day 1:** Liquid Clustering vs. Partitioning/Z-Order
- **Day 2:** Delta Optimization (Deletion Vectors, CDF)
- **Day 3:** System Tables & Query Profiler
- **Day 4:** Monitoring with Event Logs & SQL Alerts
- **Day 5:** Troubleshooting (Spark UI, Cluster Logs)

### Week 5: Advanced Modeling & Final Review
- **Day 1:** Dimensional Modeling for Lakehouse
- **Day 2:** Scalable Data Models with Delta Lake
- **Day 3:** Review: Cost Optimization & Resource Utilization
- **Day 4:** Mock Exam 1 (Focus on Scenarios)
- **Day 5:** Mock Exam 2 & Final Gap Analysis
